Import images from amazon reviews and variations to woocommerce gallery

== Description ==
Reviews Images for Amazon (dropshipping & affiliates) allows to add images from amazon reviews and variations to your woocommerce gallery, no registration, just paste your asin and click on 
add images, the plugin will do the rest for you
This can give your customer more confidence to buy the product. this can be used by dropshipping or affiliate 

== How this work == 

[youtube https://youtu.be/UnouPMsfWUw]


You click on the plugin button on the wordpress menu, then search the product that you want to enrich with images or go to woocommerce product page and click the button 
import images, a popup will be displayed, on this popup you paste your product sku and click on load, this will load the images from amazon, you can then choose which images you want to keep 
and by a single click you import them to your product gallery 

== Installation ==
To install the plugin theShark image finder for amazon, please follow this instruction.
1. install the plugin theShark image finder for amazon from wordpress directory
2. Activate the plugin theShark image finder for amazon through the ‘Plugins’ menu in WordPress
